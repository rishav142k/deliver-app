<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

* django
  ```sh
  pip install django
  ```
* PIL
  ```sh
  pip install pillow
  ```
* All-Auth
  ```sh
  pip install django-allauth
  ```
  

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/rishav142k/deliver-app/
   ```
3. Install the required packages
   ```Python
   pip install django
   pip install pillow
   pip install django-allauth
   
   ```
4. Run `manage.py`
   ```Python
   python manage.py runserver
   ```
